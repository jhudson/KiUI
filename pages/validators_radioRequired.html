Validators

Radio Required New!

Recently at work I had to implement some form validations; there was this radio button that needed to be required. I said to myself: "Nothing easier!! I'll just use the standard required Kendo validator". Unfortunately it doesn't support radio buttons. :(

Kiui Radio Required validator extends the standard Kendo required validator and makes it work with radio buttons too.

It's not that difficult to use: just go on and put that required attribute on your <input type="radio"> element and it'll just work!

  • Html
  • Javascript
<div id="validators_radioRequired_demo">

  <form>
    <div class="control-group">
      <label class="control-label">Did you download <strong>KiUI</strong> latest version?</label>
      <div class="controls">
        <label class="radio inline">
          <input type="radio" name="answer" value="N" required /> No
        </label>
        
        <label class="radio inline">
          <input type="radio" name="answer" value="Y" required /> Yes
        </label>
        
        <label class="radio inline">
          <input type="radio" name="answer" value="A" required /> ABSOLUTELY!!
        </label>
        
        <span class="help-inline"><span class="k-invalid-msg" data-for="answer"></span></span>
      </div>
    </div>
    
    <div class="control-group">
      <button type="button" class="btn btn-primary">Validate</button>
    </div>
  </form>

</div>

jQuery(function($) {
  
  var validator = $("#validators_radioRequired_demo form").kendoValidator().data("kendoValidator");
  
  $('#validators_radioRequired_demo form button').click(function(e) {
    e.preventDefault();
    
    if (validator.validate()) {
      $(this)
        .removeClass('btn-primary')
        .addClass('btn-success')
        .text('The form is valid! Yay!');
    }
  });
  
  validator.validate();
  
});
