Personally, I've had problems using Kendo UI Web date validator: it only works on input[type="date"] and since the DatePicker widget transforms it's <input> element to input[type="text"], you can't use the validator anymore.

Kiui Date validator can be applied to any <input> element and it just works with DatePickers. Additionally, you can also specify a custom culture that the validator will use when trying to parse the date.
