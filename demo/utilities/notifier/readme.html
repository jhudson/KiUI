The Notifier utility helps you to send notifications to the user.

Start creating an instance of Notifier using the kiui.notifier() function, it accepts a single position option that can have four values: top-left, bottom-left, bottom-right or top-right. Default is top-right.

The Notifier exposes three methods that you can use to create different kinds of notifications: error(), info() and success().

Each method accepts the following options:

  • append: when set to true, the given content will be appended to a previous notification of the same type; if such a notification does not exist, a standard notification will be created instead. (Default: false)
  • autoHide: when set to a number, the notification will be automatically hidden after the given amount of milliseconds. If set to false the notification will have to be hidden manually. (Default: false)
  • content: the html content that will be inserted inside the notification.
  • icon: the html that will be used as icon for the notification, if set to "" the icon will be hidden. (Default: every method has its specific default icon)
  • title: the notification title, if set to "" the title will be hidden. (Default: "")
  • width: the notification width in pixels. (Default: 250)
