Bindings

Bool Value

Tweet
Current view-model state:
{
    likeKiui: 
}

The boolValue binding can be used to bind a radio buttons control with a boolean variable in the view model.

It's able to support three different values: true, false and null.

  • Html
  • Javascript
<div id="bindings_boolValue_demo">
    <div class="span7">
        <form>
            <label>Do you like <strong>KiUI</strong>?</label>

            <label class="radio">
                <input type="radio" name="boolValue_demo" value="true" data-bind="boolValue: likeKiui" />
                Yes <span class="help-inline">(Be sure to check this one!! ;-)</span>
            </label>

            <label class="radio">
                <input type="radio" name="boolValue_demo" value="false" data-bind="boolValue: likeKiui" />
                No
            </label>

            <label class="radio">
                <input type="radio" name="boolValue_demo" value="null" data-bind="boolValue: likeKiui" />
                Don't know
            </label>
        </form>
    </div>

    <div class="span5">
        <h5>Current view-model state:</h5>
        <pre>
{
    likeKiui: <span data-bind="text: likeKiuiText()"></span>
}</pre>
    </div>

</div>
jQuery(function($) {
  var viewModel;

  viewModel = kendo.observable({
    likeKiui: null,
    likeKiuiText: function() {
      return kendo.stringify(this.get('likeKiui'));
    }
  });

  kendo.bind($('#bindings_boolValue_demo'), viewModel);

});
