Loader

New! Tweet

Try clicking the button below!

A loader overlay will appear over this <div> element until the async operation is completed.

Current view-model state:
{
    ongoingOperation: 
}

The loader binding can be used to automatically show or hide a loader overlay on a DOM element based on the value of a boolean property defined in the view model.

It's useful when you need to notify the user of an ongoing asynchronous operation.

  • Html
  • Javascript
<div id="bindings_loader_demo">
    <div class="span7">
        <div class="alert alert-info" data-bind="loader: ongoingOperation">
            <p>
                <strong>Try clicking the button below!</strong>
            </p>
            <p>
                A loader overlay will appear over this &lt;div&gt; element until the async operation is completed.
            </p>
            <p>
                <button class="btn" data-bind="click: startAsyncOperation">Start async operation</button>
            </p>
        </div>
    </div>

    <div class="span5">
        <h5>Current view-model state:</h5>
        <pre>
{
    ongoingOperation: <span data-bind="text: ongoingOperationText()"></span>
}</pre>
    </div>

</div>
jQuery(function($) {
  var viewModel;

  viewModel = kendo.observable({
    ongoingOperation: false,

    startAsyncOperation: function() {
      var that = this;

      this.set('ongoingOperation', true);

      // fake async operation
      setTimeout(function() {
        that.set('ongoingOperation', false);
      }, 3000);
    },
    
    ongoingOperationText: function() {
      return kendo.stringify(this.get('ongoingOperation'));
    }
  });

  kendo.bind($('#bindings_loader_demo'), viewModel);

});
